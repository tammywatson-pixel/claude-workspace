# ElevIQ — Founder Workspace

Central hub for ElevIQ planning, execution, and tracking.

## Directories

| Folder          | Purpose                                                  |
|-----------------|----------------------------------------------------------|
| `strategy/`     | Vision, roadmap, OKRs, market positioning, and GTM plans |
| `pilots/`       | Pilot programs — scopes, status, results, and learnings  |
| `partnerships/` | Partner leads, agreements, and relationship notes        |
| `grants/`       | Grant opportunities, applications, and award tracking    |
| `notes/`        | Meeting notes, ideas, and scratch space                  |
